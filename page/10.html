Gentoo Linux下Opera的字体设置 六月 15th, 2009

  1. 地址栏输入：opera:config ， 将Enable core X fonts打钩；而Enable Xft Fonts后面的钩去掉，然后重启Opera（如果字体仍然混乱，则把上面的两个打钩交替一下）
  2. 修改文件 /opt/opera/share/opera/ini/font.ini 【就是这个文件控制着那个令人非常头疼的，Opera 的字体选择机制】

在 [matches] 节中合适的地方添加以下内容：

family:WenQuanYi Bitmap Song=excellent
;完全禁用日文、韩文字体
family:Kochi Mincho | Baekmuk Dotum=blacklist
family:mincho|gothic=blacklist
family:kochi*=blacklist
family:baekmuk*=blacklist
;或者在需要时才调用
family:Kochi Mincho | Baekmuk Dotum=avoid family:mincho|gothic=avoid family:kochi*=avoid family:baekmuk*=avoid

其中，=后面的blacklist、excellent、avoid等等分别描述的是opera对该字体选择的优先级，excellent是优先选择，avoid是只有需要时才选择，blacklist是禁用，不予选择

解决Gentoo下Mplayer播放rmvb问题 六月 12th, 2009

首先，安装Mplayer的时候需要添加Real支持，默认情况下，real的USE是(-real)，也就是不支持real（据说因为安全性的原因，官方portage去掉了real支持），不能添加，所以必须先以root权限执行下列命令：

mkdir -p /etc/portage/profile echo media-video/mplayer -real >> /etc/portage/profile/package.use.mask echo media-libs/win32codecs -real >> /etc/portage/profile/package.use.mask ACCEPT_KEYWORDS="~x86" USE="real" emerge mplayer

安装完成后，如果Mplayer播放rmvb出现

Win32 LoadLibrary failed to load: drvc.dll, /usr/lib/codecs/drvc.dll, /usr/lib/win32/drvc.dll, /usr/local/lib/win32/drvc.dll Error loading dll ERROR: Could not open required DirectShow codec drvc.dll.

之类的错误，则需要下载以下两个文件：

drv43260.dll http://cid-ee616f2dc1c3f48b.skydrive.live.com/self.aspx/Share/drv43260.dll
drvc.dll http://cid-ee616f2dc1c3f48b.skydrive.live.com/self.aspx/Share/drvc.dll

下载完后复制到 /usr/lib/win32 目录下，即可解决

解决Nvidia96.43.11显卡驱动在Linux2.6.30内核下的编译问题 六月 11th, 2009

出现的问题

在Linux2.6.30内核下编译Nvidia 96.43.11显卡驱动时出现如下错误：

echo " ERROR: Kernel configuration is invalid."; echo " include/linux/autoconf.h or include/config/auto.conf are missing."; echo " Run 'make oldconfig && make prepare' on kernel src to fix it

这时候即使运行了

make oldconfig && make prepare

也无济于事

解决办法

到下面的地址下载一个patch： http://leigh123linux.fedorapeople.org/pub/patches/nvidia-96.43.11.patch

然后将这个patch和NVIDIA-Linux-x86-96.43.11-pkg1.run文件放到同一个目录下，cd到这个目录，在未启动X server的情况下（或者先kill掉X）再运行命令：

./NVIDIA-Linux-x86-96.43.11-pkg1.run --apply-patch nvidia-96.43.11.patch

以上在Gentoo Linux kernel 2.6.30下测试通过，适用于x86版本，x86_64版本以此类推。
转载请注明：作者 Kder 出处 http://kder.yo2.cn/nvidia-driver-solution-for-linux-2630.html
参考了帖子：http://www.nvnews.net/vbulletin/showthread.php?t=133990

我在Linux下常用的软件 六月 11th, 2009

　一个简单的列表，以后会不时更新：
  • Fetion 也就是libfetion
  • QQ ：官方Linux版 http://im.qq.com/qq/linux/download.shtml
  • Opera / Firefox / Chrome：浏览器，必备
  • Mirage：一个基于Python的小巧功能齐全的图像查看器http://mirageiv.berlios.de/download.html
  • Mousepad ：轻巧的文本编辑器
  • xterm：Xorg自带的终端，没什么特殊要求，够用就行
  • Zsh：终极shell ，可参考http://linuxtoy.org/archives/zsh.html
  • VirtualBox ：虚拟机，做实验用的
  • Vim /Scite /TextAdept ：高级文本编辑器，每个都很有名，不必介绍了
  • FoxitReader：著名的轻巧而强大的PDF查看器，官方linux版：http://www.foxitsoftware.com/pdf/desklinux/

把墙搬回家——“绿坝”事件 六月 10th, 2009

工信部日前下发通知，要求7月1日后在中国境内生产销售和进口的计算机将预装“绿坝—花季护航”绿色上网过滤软件。

看到这条新闻，我不由得想到防火长城给我带来的种种不便，“墙”也是我从blogger转移阵地到yo2来的重要原因之一。

如今，墙已经推到了自家门口。虽然现有的机子和自己的组装机不受影响，品牌机也可以将其卸载，但是总的感觉还是，非常不爽。为未成年人创造一个良好的网络环境本无可厚非，但是如何在保护青少年和保护成年人的隐私和自由这两者之间找到一个平衡，工信部应该好好考虑一下。
