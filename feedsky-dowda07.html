遭遇Feedsky宕机 十二月 2nd, 2009

　　11月29日上午，突然觉得豆瓣的Blog没有正常更新，进而发现我博客的Feed不能访问了，一查才知道，原来Feedsky宕机了。 　　我又试了一下： 　　Ping不通：
ping feedsky.com Ping request could not find host feedsky.com. Please check the name and try again.
　　域名解析不了：
nslookup feedsky.com timeout
　　网页打不开（域名无法解析）：
The requested URL could not be retrieved While trying to retrieve the URL: http://www.feedsky.com/ The following error was encountered: Unable to determine IP address from host name for feed.feedsky.com The dnsserver returned: Server Failure: The name server was unable to process this query. This means that: The cache was not able to resolve the hostname presented in the URL. Check if the address is correct.
　　通过搜索了解到，似乎06年左右Feedsky“经常”宕机，近来还没发现。没想到我的Feedsky没用多久（两个月不到吧），就遇到宕机了。为了以防万一，Kder又申请了一个Feedburner的Feed： http://feeds.feedburner.com/kderlin 与本blog同步更新，欢迎订阅。 PS：Feedburner有被和谐过的记录，目前可以访问，不知以后会否再次……
