Notepad2乱码问题 九月 3rd, 2009

对应版本的解决办法如下：

Notepad2 4.0 以前版本（ 3.x）：

在菜单中：Customize Schemes - Default Text - Default Style - Style Settings里面，把charset:0;去掉即可

Notepad2 4.0版本（目前为4.0.22）：

在菜单中：File - Encodings - Default - 选中 Skip automatic Unicode detection，否则有些中文txt文件会被识别为韩文，导致乱码
